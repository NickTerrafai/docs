---
title: "Getting an API Key"
---

To make requests to the Terrafai API you must have an API key. This API key is used to authenticate requests made to the system and to identify who is making the requests to accurately record usage information.

<Steps>
  <Step title="New Step">
    When you log into the system you will land on the dashboard page. From the left hand menu click the API Keys menu item.

    <img
      src="/images/Picture1.png"
      alt="Picture1"
      title="Picture1"
      style={{ width:"42%" }}
    />
  </Step>
  <Step title="New Step">
    Then, look over to the right and click the "Create New Key" button.

    <img
      src="/images/Picture2.png"
      alt="Picture2"
      title="Picture2"
      style={{ width:"71%" }}
    />
  </Step>
  <Step title="New Step">
    A pop up will then open. Enter the name you would like to call your API key and then click the "Create Key" button.

    <img
      src="/images/Picture3.png"
      alt="Picture3"
      title="Picture3"
      style={{ width:"90%" }}
    />
  </Step>
  <Step title="New Step">
    You will then have the opportunity to save your key. Press the "Copy" button to copy the newly created API key to your clip board and save it somewhere safe. You can then click the "Done". You have no finished creating a new API key, ready for use.

    <img
      src="/images/Picture4.png"
      alt="Picture4"
      title="Picture4"
      style={{ width:"90%" }}
      className="mr-auto"
    />
  </Step>
</Steps>
